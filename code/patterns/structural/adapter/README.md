# Adapter

Адаптер (Adapter) - это структурный паттерн проектирования, который позволяет объектам с несовместимыми интерфейсами 
работать вместе. Он создает адаптер, который выступает в качестве посредника между двумя объектами, 
преобразуя интерфейс одного объекта в интерфейс, ожидаемый другим объектом.
